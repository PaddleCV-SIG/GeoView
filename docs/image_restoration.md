# 图像复原功能使用说明

## 功能描述

对存在噪声、模糊等不同类型失真的遥感影像进行复原，或是对低分辨率影像进行超分辨率重建，输出修复或重建后的影像。

## 使用方法

点击“上传”按钮，选择单张图片或包含多张图片的文件夹并确认。
