# 在线地图使用说明

## 功能描述

提供在线百度地图服务，允许用户通过地图定位、截取图片。支持目标检测、地物分类、场景分类和图像复原任务。

## 使用方法

### 基本使用

在搜索框输入位置信息，选择“搜索结果”，定位到感兴趣地点。可在地图右上角设置地图模式（混合地图模式和卫星地图模式）。如图：
